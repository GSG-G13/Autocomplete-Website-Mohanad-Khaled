# Countries Weather App

### This project is a simple search engine, which displays the weather for the country entered in the search field.

---

## Features:

- Search by Name.
- Auto Complete Search.
- Show the Search result.

---

## Technology's

- HTML
- CSS
- JavaScript
- XHR
- Node JS
